# Calculadora Android

Un proyecto de calculadora simple para Android, desarrollada nativamente para Android utilizando **Kotlin**. El diseño sigue los lineamientos de **Material Design 3 (Material You)**, asegurando una interfaz limpia y adaptable, alineada con el estilo de Android 16.

## Características

* **Operaciones Básicas:** Suma, resta, multiplicación y división.
* **Interfaz Moderna:** Botones grandes y accesibles con retroalimentación táctil.
* **Manejo de Errores:** Prevención de división por cero.
* **Texto Dinámico:** El tamaño de los números se ajusta automáticamente para evitar cortes.

## Tecnologías Utilizadas

* **Lenguaje:** [Kotlin](https://kotlinlang.org/)
* **Entorno:** Android Studio
* **Diseño:** XML con Material Components (Material 3)
* **Vinculación de Vistas:** ViewBinding
* **Layout:** ConstraintLayout y GridLayout

## Cómo ejecutar el proyecto

1.  Descarga o clona este repositorio.
2.  Abre la carpeta del proyecto en **Android Studio**.
3.  **Espera la Sincronización (Build):** Android Studio detectará el proyecto. Espera a que la barra de progreso de "Gradle Sync" termine.
4.  **Archivo `local.properties`:** Este archivo **no** se incluye en el repositorio, ya que es específico de tu computadora. Android Studio lo **creará automáticamente** por ti durante la primera sincronización, apuntando a tu SDK de Android.
5.  Conecta tu dispositivo o inicia un emulador.
6.  Presiona el botón **Run (▶)**. Esto compilará (**build**) e instalará la app.

## Requisitos

* Android Studio Iguana o superior.
* SDK Mínimo: API 34 (Android 14.0).
* SDK Objetivo: API 36.0.
