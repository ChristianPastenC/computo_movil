# Calculadora iOS en SwiftUI

Un proyecto de calculadora simple para iOS, construido completamente con SwiftUI. El objetivo es demostrar una arquitectura limpia (MVVM) y una interfaz de usuario moderna con un efecto de "Liquid Glass" (Glassmorphism).

## Características

* **Operaciones Básicas:** Suma, resta, multiplicación y división.
* **Diseño Moderno:** Interfaz con efecto "Glassmorphism" (transparencia y desenfoque).
* **Historial de Expresión:** Muestra la operación actual o la expresión anterior (ej. "10 + 5").
* **Resaltado de Operación:** El botón de la operación activa se invierte para una mejor UX.
* **Lógica Inteligente:** Diferencia entre "AC" (All Clear) y "C" (Clear) para borrar.

## Stack Tecnológico

* **Lenguaje:** Swift
* **Framework de UI:** SwiftUI
* **Arquitectura:** MVVM (Model-View-ViewModel)
* **Entorno:** Xcode 15+

## Cómo Ejecutar

1.  Clona el repositorio en tu máquina local:
    ```bash
    git clone https://github.com/ChristianPastenC/computo_movil.git
    cd Calculadora
    ```
2.  Navega a la carpeta del proyecto y abre el archivo `.xcodeproj` en Xcode.
3.  Selecciona un simulador de iPhone (ej. iPhone 15 Pro).
4.  Presiona el botón **Run** (►) para compilar y ejecutar la aplicación.
